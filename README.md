# aplicacion-web

Crea una PWA de turismo de viaje a Almería (España)

-Señala los puntos de interes
-Inserta imágenes de picsum.photo
-Guarda la infomración de los lugares de interés en un archivo llamado lista.json (añade los lugares que consideres)
-Añade filtros de texto, por tipo de lugares (cultura, museos, restaurantes, monumentos históricos, recintos deportivos)
- Añade las siguientes vistas: lista, tarjetas, mapa (mapa de open maps), opción de ruta en google maps
- En la ficha del lugar a visitar añade una opción para enviar este lugar a un amigo
- Añade otra pestaña con la información meteorológica de Almería de hoy y los siguientes siete días a partir de una API pública con la información metorológica
